Introduction

нҳ„мһ¬ л§ҺмқҖ RL л…јл¬ёл“Өмқҙ лӮҳмҷ”кі  мқҙл“ӨмқҖ low-levelмқҳ н–үлҸҷмқҳ кІ°кіјлЎң high-levelмқҳ н–үлҸҷмқ„ мқҙлҒҢм–ҙлӮҙкі  мһҲм§Җл§Ң мқҙ low, high level н–үлҸҷмқҳ termмқҙ кёём–ҙм§„лӢӨл©ҙ л¬ём ңк°Җ л°ңмғқн•ңлӢӨ. мҰү sparse reward нҳ•нғңк°Җ лҗңлӢӨл©ҙ л¬ём ңк°Җ л°ңмғқн•ңлӢӨ. к·ёлһҳм„ң мқҙ л…јл¬ёмқҖ deep recurrent neural network кө¬мЎ°мқё STRategic Attentive Writer(STRAW)лқјлҠ” кө¬мЎ°лҘј м ңм•Ҳн•ңлӢӨ. мқҙ кө¬мЎ°лҠ” macro actionмқ„ л°°мҡё мҲҳ мһҲлҠ” кө¬мЎ°мқҙлӢӨ. multi-step actionplanмқ„ к°Җм§Җкі  мһҲкі  мЈјкё°м ҒмңјлЎң observationкіј planм—җ commitн•ҳм—¬ replanningмқ„ н•ңлӢӨ.

StrawлҠ” л‘җк°Җм§Җ лӘЁл“Ҳмқ„ к°Җм§Җкі  мһҲлӢӨ. мІ«лІҲм§ёлҠ” action plan лӘЁл“ҲлЎң macro-actionsмқ„ л§Ңл“ңлҠ” лӘЁл“ҲмқҙлӢӨ. л‘җлІҲм§ёлҠ” commit-planмқ„ к°Җм§Җкі  мһҲкі  мқҙ лӘЁл“ҲмқҖ м–ём ң macro actionмқ„ лҒқлӮҙкі  action planмқ„ м—…лҚ°мқҙнҠён•ҳлҠ” м§Җ кІ°м •н•ҳлҠ” лӘЁл“ҲмқҙлӢӨ. мқҙ л‘җ лӘЁл“Ҳмқ„ м—…лҚ°мқҙнҠён•ҳкё° мң„н•ҙ attentive writing techniqueмқ„ мӮ¬мҡ©н•ңлӢӨ. мқҙлҘј нҶөн•ҙ networkк°Җ observationмқҳ м–ҙл–Ө л¶Җл¶„мқ„ лҙҗм•ј мң мҡ©н•ң м •ліҙлҘј м–»мқ„ мҲҳ мһҲмқ„ м§Җ м•ҢкІҢ н•ҙмЈјлҠ” кё°мҲ мқҙлӢӨ.

The state of the network

t мӢңк°„м—җм„ң networkмқҳ stateлҠ” \(A^t \in R^{A \times T}\)мҷҖ \(c^t \in R^{1 \times T}\)лЎң мқҙлЈЁм–ҙм ёмһҲлӢӨ. \(A^t\)лҠ” action planмқҙлӢӨ. stepмқҙ м§ҖлӮ л•Ңл§ҲлӢӨ \(A^t\)лҘј shiftн•ҳкі  мғҲлЎңмҡҙ acion planмқ„ м¶”к°Җн•ңлӢӨ. \(c^t\)лҠ” commit planмңјлЎң м–ём ң replanningмқ„ н• м§Җ кІ°м •мқ„н•ңлӢӨ. м•„лһҳлҠ” STRAWмқҳ schematic illustrationмқ„ ліҙм—¬мӨҖлӢӨ.

Schematic illustration fo STARW

Attentive planning

н•ҳлӮҳмқҳ к°Җм •мңјлЎң н•ң observationмқҙ sequential actionмқ„ м–»кё° мң„н•ҙ м¶©л¶„н•ң лҚ°мқҙн„°лҘј к°Җм§Җкі  мһҲлӢӨлҠ” м җмқҙлӢӨ. к·ёлҹ¬лҜҖлЎң networkлҠ” мөңк·ј observationмқҙ мӣҗн•ҳлҠ” н–үлҸҷмқ„ мң„н•ҙм„ң м¶©л¶„нһҲ лҚ°мқҙн„°лҘј к°Җм§Җкі  мһҲм–ҙм•јн•ҳлҜҖлЎң planмқҳ мӨ‘мҡ”н•ң л¶Җл¶„л§Ң лҙҗм•јн•ңлӢӨ. мқҙ л…јл¬ём—җм„ңлҠ” мқҙлҘј мң„н•ҙ attentive modelмқ„ мӮ¬мҡ©н•ңлӢӨ. ліҙнҶө мқҙлҜём§Җ generationм—җм„ң мӮ¬мҡ©н•ҳлҚҳ л°©лІ•мңјлЎң мқҙ л…јл¬ём—җм„ңлҠ” DRAWлқјлҠ” л…јл¬ёмқҳ м•„мқҙл””м–ҙлҘј м°Ёмҡ©н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ.

Gaussian filterмқҳ arrayлҘј planм—җ м Ғмҡ©н•ҳкі  patch нҒ¬кё°лЎң л°”кҫјлӢӨ. нҒ° Action spaceм—җм„ң м§‘мӨ‘н•ҳм—¬ мғҲлЎӯкІҢ л§Ңл“ңлҠ” кІғмқҙлӢӨ. AлҘј к°ҖлҠҘн•ң action мҲҳ, KлҘј patch нҒ¬кё°лқјкі  н•ҳмһҗ. \(K \times A\) нҒ¬кё°мқҳ 1D gaussain filterлҘј мӮ¬мҡ©н•ҳкІҢлҗңлӢӨ. \(\psi\)лҘј grid position, stride л“ұкіј к°ҷмқҖ attention parametersлқјкі  н•ңлӢӨл©ҙ attention operationsмқ„ м•„лһҳмҷҖ к°ҷмқҙ м •мқҳн•ңлӢӨ.

Write operationмқҖ patchмҷҖ attention parametersлҘј inputмңјлЎң л°ӣлҠ”лӢӨ. Read operationмқҖ action planкіј attention parametersлҘј inputмңјлЎң л°ӣлҠ”лӢӨ. мһҗм„ён•ң л¶Җл¶„мқҖ лӢӨмқҢ sectionмқ„ ліҙл©ҙ лҗ  кІғ к°ҷлӢӨ.

Action-plan update

\(z_t\) feature represeationмқҙлқјкі  н•  л•Ң \(z_t\)мҷҖ \(g_t\)лҘј мқҙмҡ©н•ҙм„ң STRAWлҠ” plan updateлҘј н•ңлӢӨ. м•„лһҳлҠ” action-plan updateмқҳ algorithmмқҙлӢӨ.

Algorithm1

read л¶Җл¶„мқ„ ліҙл©ҙ \(A^{t-1}\)кіј attention parameter inputмқ„ л°ӣм•„м„ң action planмқ„ мғқм„ұн•ҳлҠ” кІғмқ„ ліј мҲҳ мһҲкі  мқҙл ҮкІҢ мғқкёҙ action planкіј \(z_t\)лҘј мқҙмҡ©н•ҳм—¬ patchлҘј мғқм„ұн•ҳм—¬ write operationмқ„ м§„н–үн•ҳлҠ” кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

Commitment-plan update

commitment-plan updateлҠ” \(c_t\)лҘј м°ёкі н•ҳм—¬ \(c^{t-1}\)мқҙ 0мқј кІҪмҡ° shiftлҘј н•ҳкі  м•„лӢҗ кІҪмҡ° мғҲлЎң replanningн•ҳм—¬ commit planмқ„ updateн•ңлӢӨ.

Learning

STRAWмқҳ loss functionмқҖ м•„лһҳмҷҖ к°ҷлӢӨ. \(L^out\)мқҖ A3Cмқҳ loss functionмқҙлӢӨ. action planмқҳ rawлҘј мқҙмҡ©н•ҳм—¬ value function estimationн•ҳмҳҖлӢӨ. к·ёлҰ¬кі  л‘җлІҲм§ё мӢқмқҖ latent distributionкіј priorм—җ лҢҖн•ң kl divergence мӢқмңјлЎң нҸүк·  0, л¶„мӮ° 1 мқё prior gaussian distributionм—җ л”°лҘҙлҸ„лЎқ latent distributionмқ„ л§Ңл“ лӢӨ. м„ёлІҲм§ё мӢқмқҖ termмқҖ re-planningкіј commitmentм—җ кҙҖл Ёлҗҳм–ҙ мһҲлӢӨ.

Experiment

мқҙ л…јл¬ёмқҖ 2D mazeлҘј мқҙмҡ©н•ҳм—¬ мӢӨн—ҳн•ҳмҳҖлӢӨ. кё°ліём Ғмқё fully connected layerлЎң мқҙлЈЁм–ҙм§„ л°©лІ•кіј LSTM, STRAW, STRAWeлҘј л№„көҗн•ҳмҳҖлӢӨ. STRAWeлҠ” STRAWм—җ gaussian noiseлҘј лҚ”н•ҳм—¬ explorationмқ„ к¶ҢмһҘн•ҳлҠ” л°©лІ•мқҙлӢӨ. мӢӨн—ҳ кІ°кіјлҘј ліҙл©ҙ optimal кІ°кіјмқё лӢӨмқөмҠӨнғҖ м•Ңкі лҰ¬мҰҳ лҢҖл№„ STRAWк°Җ көүмһҘнһҲ мһҳ л”°лқјк°ҖлҠ” кІғмқ„ нҷ•мқён•  мҲҳ мһҲкі  LSTMмқҙ к°җмҶҢнҸӯмқҙ мӢ¬н•ң кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.
